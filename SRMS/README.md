# 📌 Project Overview
I built a Java Console-Based Student Record Management System that allows users to add, view, update, and delete student records directly through the console. The system continuously accepts user input until the user decides to exit.

# 🔨 What I Did:
✅ Designed a menu-driven CRUD system using Java.  
✅ Created a `Student` class with encapsulation principles.  
✅ Used `ArrayList` to store and manage student records dynamically.  
✅ Implemented options to add, view, update, and delete student records.  
✅ Provided real-time user interaction using Scanner for input.  
✅ Followed clean, modular, and easy-to-understand coding practices.

# ⚙️ Tools Used:
- Java (JDK)
- VS Code / Command Line
